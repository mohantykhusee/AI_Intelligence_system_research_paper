# 🧠 AI Intelligence System Research Paper

An AI-powered research paper analysis and search system that helps users efficiently explore academic papers using Natural Language Processing (NLP). The project includes intelligent search, keyword extraction, text summarization, and exploratory data analysis (EDA).

---

## ✨ Features

- 🔍 AI-powered Research Paper Search
- 📑 Automatic Research Paper Summarization
- 🏷️ Keyword Extraction using NLP
- 📊 Exploratory Data Analysis (EDA)
- 📂 Dataset Documentation
- ⚡ Easy-to-use Jupyter Notebooks

---

## 📁 Repository Structure


## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Jupyter Notebook

### Installation

```bash
git clone https://github.com/yourusername/AI_Intelligence_System_Research_Paper.git

cd AI_Intelligence_System_Research_Paper

pip install -r requirements.txt
```

---

## ▶️ Usage

1. Open Jupyter Notebook.
2. Run `EDA.ipynb` for data exploration.
3. Run `Search_Engine.ipynb` to:
   - Search research papers
   - Extract keywords
   - Generate summaries
